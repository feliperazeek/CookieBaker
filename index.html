CookieBaker by feliperazeek

CookieBaker is a simple Java library to read and write cookies in POJO form to avoid string splits and joins all over your codebase. CookieBaker also provides HMAC support off-shelve, currently using SHA1 by default.

Dependencies

JUnit, Servlet API, Jackson, Spring Test

Install

git clone git@github.com:feliperazeek/CookieBaker.git cd CookieBaker mvn install
